## Overview

The `CNN` class provides a simple interface for training convolutional neural networks. It supports 30+ architectures out of the box, plus all 800+ models from the timm library.

## Basic Usage

```python
from kitepy import CNN

# Train ResNet-18 on CIFAR-10
model = CNN("resnet18")
model.train("cifar10")
```

## Supported Architectures


## Pretrained Weights

Load ImageNet pretrained weights for transfer learning:

```python
model = CNN("resnet50", pretrained=True)
model.train("my_dataset")  # Fine-tune on your data
```

Pretrained weights are available for most popular architectures.

## Custom Number of Classes

Adapt the model for your number of classes:

```python
# CIFAR-100 has 100 classes
model = CNN("resnet50", num_classes=100)
model.train("cifar100")

# Binary classification
model = CNN("efficientnet_b0", num_classes=2)
model.train("my_binary_dataset")
```

## Transfer Learning

### Freeze and Fine-Tune

```python
from kitepy import CNN

# Load pretrained model
model = CNN("resnet50", pretrained=True, num_classes=10)

# Phase 1: Freeze backbone, train only classifier
model.freeze_backbone()
model.train("my_dataset", epochs=10, lr=0.01)

# Phase 2: Unfreeze and fine-tune entire model
model.unfreeze()
model.train("my_dataset", epochs=50, lr=0.0001)
```

### Gradual Unfreezing

```python
# Unfreeze last N layers only
model.unfreeze(layers=10)  # Unfreeze last 10 layers
```

## Custom CNN Architecture

Pass your own PyTorch model:

```python
import torch.nn as nn

# Define custom architecture
custom_model = nn.Sequential(
    nn.Conv2d(3, 64, kernel_size=3, padding=1),
    nn.BatchNorm2d(64),
    nn.ReLU(),
    nn.MaxPool2d(2),
    
    nn.Conv2d(64, 128, kernel_size=3, padding=1),
    nn.BatchNorm2d(128),
    nn.ReLU(),
    nn.MaxPool2d(2),
    
    nn.Conv2d(128, 256, kernel_size=3, padding=1),
    nn.BatchNorm2d(256),
    nn.ReLU(),
    nn.AdaptiveAvgPool2d(1),
    
    nn.Flatten(),
    nn.Linear(256, 10)
)

# Use with kitepy
model = CNN(custom_model)
model.train("cifar10")
```

## Inference

### Single Image

```python
import torch
from PIL import Image
from torchvision import transforms

# Load trained model
model = CNN.load("my_model.pt")

# Preprocess image
transform = transforms.Compose([
    transforms.Resize(256),
    transforms.CenterCrop(224),
    transforms.ToTensor(),
    transforms.Normalize([0.485, 0.456, 0.406], [0.229, 0.224, 0.225])
])

image = Image.open("cat.jpg")
input_tensor = transform(image).unsqueeze(0)

# Predict
output = model.predict(input_tensor)
predicted_class = output.argmax(dim=1).item()
print(f"Predicted class: {predicted_class}")
```

### Batch Inference

```python
# Process multiple images
outputs = model.predict(batch_of_images)
predictions = outputs.argmax(dim=1)
```

## Save and Load

```python
# Train and save
model = CNN("resnet18")
model.train("cifar10", epochs=10)
model.save("my_resnet18.pt")

# Load later
loaded_model = CNN.load("my_resnet18.pt")
loaded_model.evaluate("cifar10")
```
