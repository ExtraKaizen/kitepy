## Overview

Training large-scale models on ImageNet.

## Basic Training

```python
from kitepy import CNN

model = CNN("resnet50", pretrained=False)
model.train("imagenet", epochs=90)
```

## Optimized Recipe

```python
model = CNN("resnet50")

model.train(
    "imagenet",
    epochs=90,
    batch_size=256,
    lr=0.1,
    optimizer="sgd",
    optimizer_kwargs={"momentum": 0.9},
    scheduler="multistep",
    scheduler_kwargs={"milestones": [30, 60, 80], "gamma": 0.1},
    weight_decay=1e-4,
    warmup_steps=5000,
)
```

## Vision Transformer on ImageNet

```python
from kitepy import Transformer

model = Transformer("vit_base", pretrained=True)

model.train(
    "imagenet",
    epochs=300,
    batch_size=1024,  # Use gradient accumulation if needed
    lr=1e-3,
    optimizer="adamw",
    scheduler="cosine",
    weight_decay=0.3,
    warmup_steps=10000,
    augmentation="randaugment",
)
```

## Memory Tips

```python
model.train(
    "imagenet",
    batch_size=64,
    gradient_accumulation_steps=4,  # Effective: 256
    gradient_checkpointing=True,
    mixed_precision=True,
)
```
