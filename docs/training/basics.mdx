## Overview

kitepy handles the entire training loop for you. This page covers the essential training features.

## Basic Training

The simplest training command:

```python
from kitepy import CNN

model = CNN("resnet18")
model.train("cifar10")
```

This uses intelligent defaults:
- 10 epochs
- Batch size 32
- AdamW optimizer
- Cosine learning rate schedule
- Mixed precision training (if GPU available)
- Automatic checkpointing

## Training Parameters

Override any default:

```python
model.train(
    data="cifar10",
    
    # Duration
    epochs=50,
    
    # Batch size
    batch_size=128,
    
    # Learning rate
    lr=0.001,
    
    # Optimizer
    optimizer="adamw",
    weight_decay=0.05,
    
    # Scheduler
    scheduler="cosine",
    warmup_steps=500,
)
```

## Epochs and Steps

### Epoch-based Training

```python
model.train("cifar10", epochs=100)
```

### Step-based Training

```python
model.train("imagenet", max_steps=100000)  # Overrides epochs
```

## Batch Size

### Simple Batch Size

```python
model.train("cifar10", batch_size=64)
```

### Gradient Accumulation

Simulate larger batches with limited GPU memory:

```python
model.train("cifar10",
    batch_size=32,
    gradient_accumulation_steps=4,  # Effective batch size: 128
)
```

This is especially useful for Vision Transformers and large models.

## Learning Rate

### Fixed Learning Rate

```python
model.train("cifar10", lr=0.001)
```

### Learning Rate Finding

Find the optimal learning rate automatically:

```python
suggested_lr = model.find_lr("cifar10")
print(f"Suggested LR: {suggested_lr}")  # e.g., 0.00234

model.train("cifar10", lr=suggested_lr)
```

### Warmup

Gradually increase learning rate at the start:

```python
model.train("imagenet",
    lr=0.1,
    warmup_steps=5000,  # Number of steps
)

# Or as fraction of total steps
model.train("imagenet",
    lr=0.1,
    warmup_steps=0.1,  # First 10% of training
)
```

## Optimizers

### AdamW (Default)

```python
model.train("cifar10", optimizer="adamw")
```

### SGD with Momentum

```python
model.train("cifar10",
    optimizer="sgd",
    optimizer_kwargs={"momentum": 0.9}
)
```

### Other Optimizers

```python
# Adam
model.train("cifar10", optimizer="adam")

# RMSprop
model.train("cifar10", optimizer="rmsprop")

# NAdam
model.train("cifar10", optimizer="nadam")

# RAdam
model.train("cifar10", optimizer="radam")

# Lion (requires pip install lion-pytorch)
model.train("cifar10", optimizer="lion")
```

### Custom Optimizer

```python
import torch.optim as optim

# Pass optimizer class
model.train("cifar10", optimizer=optim.Adagrad)

# Or optimizer instance
optimizer = optim.SGD(model.model.parameters(), lr=0.01, momentum=0.9)
model.train("cifar10", optimizer=optimizer)
```

## Learning Rate Schedulers

### Cosine Annealing (Default)

```python
model.train("cifar10", scheduler="cosine")
```

### Linear Decay

```python
model.train("cifar10", scheduler="linear")
```

### Step Decay

```python
model.train("cifar10",
    scheduler="step",
    scheduler_kwargs={"step_size": 30, "gamma": 0.1}
)
```

### OneCycle

```python
model.train("cifar10", scheduler="onecycle")
```

### Warmup + Cosine

```python
model.train("cifar10",
    scheduler="warmup_cosine",
    scheduler_kwargs={"warmup_steps": 1000}
)
```

### No Scheduler

```python
model.train("cifar10", scheduler="none")
```

## Mixed Precision Training

Automatically use mixed precision for faster training:

```python
model.train("cifar10", mixed_precision=True)  # Default on GPU
```

Options:
- `True` or `"fp16"`: Use FP16 (default)
- `"bf16"`: Use BFloat16 (better numerical stability)
- `False` or `"fp32"`: Full precision

## Gradient Clipping

Prevent exploding gradients:

```python
model.train("cifar10", max_grad_norm=1.0)  # Default: 1.0
```

Set to 0 to disable:

```python
model.train("cifar10", max_grad_norm=0)
```

## Regularization

### Weight Decay

```python
model.train("cifar10", weight_decay=0.01)
```

### Dropout

```python
model.train("cifar10", dropout=0.1)
```

### Label Smoothing

```python
model.train("cifar10", label_smoothing=0.1)
```

## Checkpointing

### Automatic Checkpointing

```python
model.train("cifar10",
    checkpoint_dir="./checkpoints",
    save_every_n_epochs=5,  # Save every 5 epochs
    save_top_k=3,           # Keep best 3 checkpoints
)
```

### Resume Training

```python
model.train("cifar10", resume_from_checkpoint="./checkpoints/model_epoch5.pt")
```

## Early Stopping

Stop training when validation loss stops improving:

```python
model.train("cifar10",
    early_stopping=True,
    early_stopping_patience=5,      # Wait 5 epochs
    early_stopping_min_delta=1e-4,  # Minimum improvement
)
```

## Logging

### Log Frequency

```python
model.train("cifar10", log_every_n_steps=50)
```

### Validation Frequency

```python
model.train("cifar10",
    val_check_interval=0.5,  # Validate every half epoch
)

# Or step-based
model.train("cifar10",
    val_check_interval=1000,  # Validate every 1000 steps
)
```

## Reproducibility

### Set Seed

```python
model.train("cifar10", seed=42)
```

### Deterministic Training

```python
model.train("cifar10",
    seed=42,
    deterministic=True,  # Slower but reproducible
)
```

## Debugging

### Fast Dev Run

Test your training loop quickly:

```python
model.train("cifar10", fast_dev_run=True)  # 1 batch only
```

### Overfit Test

Check if model can overfit on small data:

```python
model.train("cifar10", overfit_batches=10)  # Overfit on 10 batches
```

### Anomaly Detection

Find NaN/Inf issues:

```python
model.train("cifar10", detect_anomaly=True)
```
